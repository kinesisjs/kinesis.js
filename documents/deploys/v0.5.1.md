## Feature & Modüller

- **İnterpolasyon modları:** `linear`, `cubic`, `geodesic`, `smooth` (3-nokta Catmull-Rom),
  `none`, `adaptive`, `CustomInterpolator` — hepsi implement.
- **Adaptive zonlar:** <500ms `none` · 500ms–8s `linear` · 8s–15s `fade` · >15s `snap`.
- **Yaşam döngüsü:** `active`/`warning`/`stale`/`completed` + Sweeper + `markCompleted()`.
- **Web Worker modu:** `worker: true` (inline blob) / `{ url }`; `WorkerTracker` proxy.
  Sınır: worker'da `CustomInterpolator` yok, `fade` → snap'e iner, `getMemoryEstimate` yok.
- **Playout buffer:** `playout: 'auto'` (son ~10 gap ölçer) / manuel — render temposunu
  ingest'ten ayırır.
- **Sanity checks:** anormal sıçrama (mesafe) → fade; keskin dönüş (>90°) → cubic.
- **Ingest throttling:** araç başına min 100ms. **Sabit bellek:** ring slot (~184 B/araç).
- **Adaptörler:** OpenLayers (trail, gap-viz, opacity, speed-band, heading) ve Leaflet (parite).
- **Wrapper:** Angular standalone directive + `kinesisTracker` factory (Signal/Observable, DestroyRef).
- **route-aware:** OSRM road-snap + LRU/segment-hash cache + detour guard.

## Güvenlik

- **Input validation (core):** id/lng/lat aralık kontrolü; non-finite heading/speed düşürülür.
- **DOM-XSS (leaflet):** `divIcon` HTML'inde `escapeAttr()` + `toFinite()`.
- **SSRF (route-aware):** OSRM baseUrl `^https?://`, `profile` encodeURIComponent, fetch timeout.
- **Supply-chain:** npm provenance + OIDC Trusted Publishing; `sideEffects:false`, `files` whitelist.
- **CI:** Actions commit-SHA pinli; release `permissions` least-privilege; timeout + concurrency.
- **Error-as-event:** public method'lar throw etmez; hatalar `error` event'ine döner.
- **Açık boşluklar:** `ci.yml` `permissions` bloğu yok; CI'da `npm audit`/Dependabot yok;
  `SECURITY.md` yok; leaflet `color` string'leri escape edilmiyor; OSRM `profile` whitelist yok.
